# The Wedding of Elcky & Randi

Invitation website for Elcky Natasya & Randi.

## Tech Stack

- React + Vite
- TailwindCSS v3.4.17
- Framer Motion
- React Icons

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run development server:

   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design

- **Colors**: Cream (#FAF7F2), Lavender (#8B7AA8), Gold (#C9A24D)
- **Fonts**: Playfair Display (Serif), Inter (Sans-serif)
